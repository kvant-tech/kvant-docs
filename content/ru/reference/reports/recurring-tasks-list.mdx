# Список повторяющихся задач

В разделе «Список повторяющихся задач» представлена информация обо всех повторяющихся задачах, созданных в системе. Раздел доступен только пользователям с разрешением «Показывать отчёт: Список ПЗ».

### Поиск и фильтрация

Для удобства предусмотрена система фильтров. Чтобы воспользоваться поиском, нажмите на фильтр «Текст», введите нужный фрагмент — и система найдёт задачи по следующим полям:

Название задачи
Описание
Ожидаемый результат
Описание доказательств

### Доступные действия

С каждой повторяющейся задачей можно выполнить следующие действия:

#### Поставить на паузу

Приостановит автоматическое создание новых коммуникаций. Под списком появится предупреждение:
«Внимание! Задача приостановлена!»
Кнопка изменится на «Запустить» — она возобновит создание задач по расписанию.

#### Просмотреть список коммуникаций

Откроется окно «Список коммуникаций», где задачи отображаются в виде таблицы с колонками:

Задача — значение из поля Название

Ожидаемый результат — значение из соответствующего поля

Крайний срок — рассчитывается от заданного времени

Этап — текущий статус:
Новые (Входящие), Принято, В работе, Выполнено (Готово)

Ответственный — аватар, фамилия и имя исполнителя

Отправитель — аватар, фамилия и имя создателя задачи
При клике на строку откроется соответствующая коммуникация.

#### Редактировать

Открывает окно редактирования повторяющейся задачи.

#### Скопировать

Создаёт копию задачи и открывает её на редактирование. Название новой задачи будет начинаться со слова «Копия».

#### Удалить

После удаления внизу экрана появится уведомление:
«Повторяющаяся задача удалена»,
а кнопки действий исчезнут.
Можно воспользоваться кнопкой «Восстановить», чтобы вернуть задачу. В этом случае появится сообщение:
«Повторяющаяся задача восстановлена»,
и все действия снова станут доступны.

### Фильтры

Для быстрого поиска нужной информации доступны следующие фильтры:

#### Текст

Введите часть названия задачи. Будут показаны все совпадающие ПЗ.

#### Кому

Выберите одну или несколько функций, которым назначены повторяющиеся задачи. Будут показаны только задачи, где указанные функции — исполнители.

#### Кроме

Показывает задачи, в которых указанные функции исключены (в поле Кроме).

#### Создатель

Отфильтровывает задачи по сотруднику, который их создал. Выберите нужное ФИО из списка.
