# Как работать с отчетом "ПО ЗАТОРАМ"

В этой статье вы узнаете, как работать с разделом отчет по заторам.

Данный раздел содержит информацию о заторах (нарушениях, которые не были исправлены в течение 2-х дней), имеющихся у сотрудников организации (отображается, если есть разрешение Показывать отчет по заторам).

### Структура отчета

Отчет состоит из:

#### Фильтр "Сотрудник" (1)

По умолчанию стоит значение "Не выбрано" - в таком случае отображается отчет по всем сотрудникам компании, при нажатии на значение "Не выбрано" появляется выпадающий список, в котором при выборе сотрудника отображается информация по данному сотруднику.

#### Выбор периода (2)

Возможность выбора периода, в который был создан первый затор:

- **Любая** (будут отображены все заторы)
- **Сегодня**
- **Вчера**
- **Неделя** (будут отображены заторы, первый из которых был создан в 7 последних дней)
- **Месяц** (будут отображены заторы, первый из которых был создан в 30 последних дней)
- **3 месяца** (будут отображены заторы, первый из которых был создан в 90 последних дней)
- **Дата** (по умолчанию устанавливается сегодняшняя дата)

Данную дату можно менять с помощью стрелок вправо и влево. При выборе даты в календаре можно выбрать как отдельный день, так и необходимый интервал.

![Фильтры отчета по заторам](/from-carrot-images/zator_1.png)

#### Содержание (3)

Где отображается ФИО сотрудника, и количество его заторов.

### Подробная информация

При нажатии на строку отчет открывается подробная информация по заторам:

#### Первое нарушение

Дата создания первого затора.

#### Передано руководителю

Сколько раз информация о заторах этого сотрудника была передана руководителю организации или хранителю системы.

#### Заторов

Количество заторов, которые сейчас есть у сотрудника.

#### Список коммуникаций

Список всех коммуникаций, по которым есть нарушения, распределенные по типам нарушений.
