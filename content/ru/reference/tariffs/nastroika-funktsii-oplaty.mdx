# Автоматическая задача на оплату тарифа

*Как назначить функцию для отправки ей автоматической задачи для оплаты тарифного плана*

1. Открываем раздел настройки.

*По умолчанию данный раздел есть только у "руководителя организации".*
*Чтобы сотрудник самостоятельно мог видеть данный раздел, необходимо на одной из его функций, в разделе "функции" включить разрешение: "Показывать меню настроек".*
*Разрешение на функцию можно запросить у руководителя организации.*

2. Нажимаем на вкладку "Дополнительные".

![Изображение 2](/from-carrot-images/tariffs_55.png)

3. В данном разделе нас с Вами интересует опция "Оплата"

Укажите функцию, которая отвечает за оплату системы (например, Бухгалтер), по умолчанию функция "Не выбрано".


![Изображение 3](/from-carrot-images/tariffs_56.png)

4. Нажимаем на раздел "Не выбрано" и выбираем из списка функций ту, которая будет у нас отвечать за оплату тарифного плана. Сохраняем внесенные изменения.

![Изображение 4](/from-carrot-images/tariffs_57.png)
