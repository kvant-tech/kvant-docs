# Редактирование регламентов на функции

Для того чтобы добавить регламент на функцию, нужно нажать кнопку Добавить, затем выбрать "Для администратора" и/или "Для исполнителей" функции.

{/* ![](/help_images_ru/add_on_function.png) */}

Далее появится бейдж с зеленой подсветкой и для сохранения изменений нужно нажать зеленую кнопку "Сохранить" внизу окна.

{/* ![](/help_images_ru/green.png) */}

Для того чтобы удалить регламент с функции, нужно нажать по крестику в бейдже, тогда подсветка бейджа станет красной и затем для сохранения изменений нужно нажать зеленую кнопку "Сохранить" внизу окна.

{/* ![](/help_images_ru/red.png) */}
