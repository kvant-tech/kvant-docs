# Способ заполнения

Данная настройка позволяет выбрать вариант заполнения статистики, т.е. каким образом будут внесены данные в статистику.

Существуют три способа заполнения:

**Ручной ввод** - выбрав данный способ, заполнение статистики осуществляется введением значения вручную. **Сумма нескольких** - выбрав данный способ, статистики будут просчитываться автоматически из данных других статистик, из которых необходимо суммировать показатели. 
Выбрать другие статистики можно путем нажатия на кнопку "Добавить".

**Подсчет баллов** - данный способ заполнения статистики осуществляется введением значения вручную, и формирует показатели, которые учитываются в баллах.
