# Список задач

Данная вкладка используется для работы со списком задач проекта - их добавления, редактирования и удаления.

Добавить задачи можно несколькими способами:

1. С помощью кнопки "Добавить задачу" - после нажатия на кнопку в списке задач появится новая строка, в которой необходимо заполнить нужные поля;
2. Нажать на серый текст "Добавить задачу…" - данная кнопка отображается, если добавлена хотя бы одна задача, после нажатия на кнопку в списке задач появится новая строка, в которой необходимо заполнить нужные поля;

![](/help_images_ru/project_task_add.png)

Задача создается в случае, если заполнены обязательные поля Название и Кому. Если данные поля не заполнены, то создается заготовка задачи, которая сохраняется в списке задач проекта, но пока не отображается ни у кого в разделе Коммуникации.

Просмотреть и редактировать созданную задача можно с помощью нажатия на иконку ![](/help_images_ru/icon_open_eye.png), просмотреть заготовку заготовку нельзя, у заготовки отображается иконка ![](/help_images_ru/icon_closed_eye.png), чтобы перейти к редактированию заготовки или задачи необходимо нажать на иконку ![](/help_images_ru/icon_edit.png), удалить задачу или заготовку из проекта с помощью можно по нажатию на иконку ![](/help_images_ru/icon_delete.png). **Примечания:**

Отправителем задачи является ответственный сотрудник (если указан в настройках) или сотрудник, который создал задачу.

В участники коммуникации нельзя добавить ответственного за проект сотрудника - он не будет отображаться в списке доступных для добавления в участники сотрудников.
