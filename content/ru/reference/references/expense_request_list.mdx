# Заявки на расход

В данном разделе осуществляется создание новых заявок на расходы от групп.

Создать новую заявку на расход от группы можно с помощью кнопки "Добавить"(1) и выбрав из списка нужную группу(2).

{/* ![](/help_images_ru/add_applications.png) */}

Все заявки делятся на 3 типа:

1. Разовая
2. Постоянная
3. Выплата сотрудникам

В разделе есть 4 этапа:

**1. Новые**:

{/* ![](/help_images_ru/stage_new.png) */}

На этап **"Новые"** попадают созданные заявки типа Разовая и Выплата сотрудникам.

**2. На оплате**:

{/* ![](/help_images_ru/stage_on_payment.png) */}

На этап **"На оплате"** попадают заявки, которые были согласованы и сумма указанная в заявке будет выплачена.

**3. Оплачено**:

{/* ![](/help_images_ru/stage_paid.png) */}

На этап **"Оплачено"** попадают заявки, которые были оплачены.

**4. Постоянные**:

{/* ![](/help_images_ru/stage_permanent.png) */}

На этап **"Постоянные"** попадают только заявки типа Постоянные. И потом из этих заявок каждый месяц в указанное число, будет создаваться Разовая заявка на этапе "Новая".

*Примечание: создавать, редактировать и перемещать заявки между этапами, могут только те участники, у кого есть разрешение "Разрешить управлять финансами".*

Для типов заявок "Разовые" и "Выплата сотрудникам" отображаются следующие данные:

1. Название заявки
2. Сумма указанная в заявке
3. Дата создания заявки
4. До какого числа нужно оплатить заявку
5. Кто создал заявку

{/* ![](/help_images_ru/expense_request.png) */}

Для типа заявок "Постоянные" отображаются следующие данные:

1. Название постоянной заявки
2. Сумма указанная в заявке
3. Дата создания заявки
4. Дата следующего создания заявки
5. Кто создал заявку

{/* ![](/help_images_ru/permanent_application.png) */}
