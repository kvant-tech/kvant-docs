# Список повторяющихся задач (ПЗ)

Отчет содержит информацию о всех повторяющихся задачах, созданных в системе.

Также можно воспользоваться поиском, нажав на "Не указано" и в появившемся поле ввести нужный текст. Поиск будет осуществляться по названию задачи, её описанию, ожидаемому результату, а также по описаниям доказательств.

С повторяющимися задачами можно выполнить следующие действия:

- поставить на паузу (1), тогда новые коммуникации не будут создаваться до тех пор пока Вы вновь не запустите её. Для запуска ПЗ нажмите на значок ![](/help_images_ru/repeat_task_play.png) у повторяющейся задачи;
- просмотреть список созданных коммуникаций по данной ПЗ (2);
- редактировать (3);
- скопировать (4) - при нажатии на кнопку откроется точная копия ПЗ, в самом начале названия задачи будет слово "Копия";
- удалить (5).
