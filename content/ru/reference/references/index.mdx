# Справочные статьи

В этом разделе собраны все справочные статьи по работе с Квантом. Здесь вы найдете подробную информацию о различных функциях и настройках системы.

## Структура справок

Справочные статьи организованы по темам:

- **Коммуникации** - создание, управление и настройка коммуникаций
- **Функции и ОФС** - работа с организационной функциональной схемой
- **Разрешения** - настройка прав доступа и разрешений
- **Настройки** - конфигурация системы
- **Отчеты** - работа с отчетами и аналитикой
- **Бизнес-процессы** - создание и управление бизнес-процессами
- **Проекты** - работа с проектами
- **И многое другое**

Выберите нужную тему из списка слева для получения подробной информации.
