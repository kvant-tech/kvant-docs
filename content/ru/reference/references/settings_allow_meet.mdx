# Тип коммуникаций "Встреча"

Настройка позволяет создавать коммуникации типа "Встреча"

**Включено** - кнопка для создания Встречи будет отображаться.

**Выключено** - кнопка для создания Встречи не будет отображаться.

**По умолчанию** - настройка выключена.
