# Условие бизнес-процесса

Типы условий Блок "Условие" отвечает за настройку критериев, по которым система будет проверять состояние задачи или бизнес-процесса перед продолжением выполнения. Существует 4 вида условий: **1. Одобрена ли задача** - проверяет, была ли одобрена указанная задача. Если задача одобрена, процесс продолжается по зеленой линии, если нет — по красной. Перед блоком Условие "Одобрена ли задача" обязательно необходимо добавлять блок "Задержки". Это нужно для того, чтобы предоставить отправителю достаточное время на принятие решения и одобрение коммуникации. Без блока "Задержки" процесс может завершиться преждевременно, не дождавшись одобрения, что приведёт к сбою в логике выполнения БП. **2. Произведен ли результат в задаче** - проверяет, был ли выполнен ожидаемый результат в задаче. При выполнении результата процесс идет по зеленой линии, в противном случае — по красной. **3. Завершен ли БП** - проверяет, завершен ли выбранный бизнес-процесс. Если БП завершен, выполнение продолжается по зеленой линии, если не завершен — по красной. **4. Свое условие** - позволяет настраивать уникальные результаты завершения задачи, по которым определяется дальнейший ход БП в зависимости от выбранного результата.
