# Доступ к коммуникациям по ссылке

Настройка нужна для ограничения доступа к коммуникации по ссылке для сотрудников, которые не являются отправителем, исполнителем или участником этой коммуникации. Доступ устанавливается общий, для всех коммуникаций компании. *(Прим. Можно изменить настройки доступа для конкретной коммуникации в карточке этой коммуникации. Это может сделать только отправитель)*.

**Включено** - коммуникации доступны по ссылке (если в настройках доступа самой коммуникации ничего не меняли) **Выключено**- коммуникации не доступны по ссылке (если в настройках доступа самой коммуникации ничего не меняли) **По умолчанию** - Включено.
