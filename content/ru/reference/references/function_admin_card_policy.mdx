# Регламенты

**Регламент** - это устав, свод правил, устанавливающий порядок работы или деятельности.

На данной вкладке есть фильтр "Распространяется", который может отфильтровать регламенты, которые распространяются на определенных получателей, а именно:

- на всех
- на администраторов
- на исполнителей.

Также Вы можете увидеть вкладки "На функцию, "Сверху", "Все". - **На функцию** - регламенты, которые распространяются на данную функцию. - **Сверху**- регламенты, которые распространяются на подразделение, т.е. на исполнителей административной функции. - **Все** - регламенты, которые распространяются на всех сотрудников компании и на всех администраторов. Создать новый регламент можно в разделе Регламенты.
