# Способ заполнения

Существует несколько способов заполнения:

**Ручной ввод** - при заполнении статистики сотрудник будет вручную вводить количество показателей в выбранной единице измерения.

**Сумма нескольких** - статистика рассчитывается автоматически, если заполнены статистики, указанные в настройках как статистики для суммирования.

**Подсчет баллов** - данный способ заполнения используется, если сложно измерить количественно в натуральных или стоимостных единицах результат работы сотрудника.

*Например, есть специалист по обеспечению рабочих мест, который отвечает за то, чтобы вся компьютерная техника и программное обеспечение работали без сбоев. Его продукт — бесперебойная работа множества разных устройств и программ: компьютеры, сервер, сетевое оборудование, электронная почта и т.д.* *Чтобы его статистика отражала весь продукт, все его составляющие должны быть измерены каким-то единым способом, что бывает непросто из-за разной значимости разных задач.* *Например, ноутбук может считаться "работающим без сбоев", если он простаивал не более часа в неделю. Часа в неделю вполне достаточно для обслуживания или оперативного устранения каких-то незначительных сбоев. А сервер компании не может быть остановлен в рабочее время более чем на 30 минут в неделю, чтобы это считалось приемлемым. При этом бесперебойная работа сервера во много раз важнее для производства продукта, чем работоспособность одного из ноутбуков сотрудников. Измерить такой продукт просто в деньгах или штуках невозможно.* *В таком случае можно использовать систему баллов, в которой в статистику добавляется определенное количество баллов за каждый результат. Баллы добавляются, если произведенный результат соответствует установленным критериям качества.* *Например, для специалиста по обеспечению рабочих мест: 3 балла, если интернет всю неделю работал без сбоев; 5 баллов, если АТС работала без сбоев; 10 баллов за каждый настроенный компьютер для сотрудника.*
