# Создавать

Здесь можно указать как будет создаваться задача из бизнес-процесса, на случайного сотрудника на функции или по очереди.

**На случайного сотрудника** - задача из бизнес-процесса будет создана на случайного сотрудника на функции.

**По очереди** - задача из бизнес-процесса будет создана по очереди на каждого сотрудника на функции.

*Например: на функции стоит 3 сотрудника: Сотрудник1, Сотрудник2, Сотрудник3. При выборе "на случайного сотрудника" задача из бизнес-процесса будет создана на случайного сотрудника на функции, а при выборе "по очереди" - в первую итерацию задача из бизнес-процесса будет создана Сотруднику1, во вторую итерацию Сотруднику2, в третью итерацию Сотруднику3, в четвертую итерацию Сотруднику1 и т.д.* ***Примечание:** пункт "Создавать" отображается только когда исполнителем задачи выбран Администратор/Исполнитель функции.*
