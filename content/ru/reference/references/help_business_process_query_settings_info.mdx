# Инструкция по работе с запросами к стороннему API

Здесь вы можете настроить HTTP-запросы к сторонним API. Данный блок позволяет отправлять запросы, указывая метод, URL, заголовки и тело запроса. **1. Настройка параметров запроса** 1. **Название для запроса** - введите название, которое поможет вам идентифицировать блок на схеме бизнес-процесса. 2. **Метод запроса** - выберите тип HTTP-запроса. Тип запроса нужно выбирать исходя из требований описанных в API сервисе, который вы подключаете. Краткое описание типов запросов: 3. **URL-адрес** - укажите URL-адрес API, на который будет отправлен запрос. 
Например, **GET запрос** *https://jsonplaceholder.typicode.com/users* - получить список пользователей **2. Настройка тела запроса (если требуется)** - в **Тело запроса** введите **Ключ** и **Значение** для каждого параметра, которые требуются для API сервиса. **3. Настройка заголовков запроса (если требуется)** - в **Дополнительные заголовки запроса** можно добавить любые необходимые заголовки. Введите **Ключ** (например, *Authorization, Content-Type*) и **Значение** для каждого заголовка. 
*Примечание: в поле **Ключ** поддерживаются только **латинские буквы**, **цифры** и **тире**.* **4. Тестирование запроса** - чтобы проверить правильность запроса, нажмите кнопку **Протестировать**. В результате вы увидите статус и данные ответа, которые помогут удостовериться, что запрос настроен корректно. ![](/help_images_ru/bp_request_action_result.png) **5. Сохранение и использование запроса** - После завершения настройки нажмите **Сохранить**. Запрос будет интегрирован в ваш бизнес-процесс и выполнится в соответствии с настроенным процессом. ![](/help_images_ru/bp_request_action.png)
