# Дополнительные

В разделе «Дополнительные» настраиваются вспомогательные процессы — отправка задач на изучение регламентов, уведомления о днях рождения, контроль оплаты и коммуникаций.

## Регламенты

Этот блок предназначен для автоматизации работы с изучением регламентов.

#### Укажите функцию контроля изучения регламентов

Данная настройка предназначена для того, чтобы указать, кто будет отправителем задачи на изучение регламентов, которая приходит сотрудникам каждую неделю, если у сотрудников есть неизученные регламенты.

#### Сколько регламентов минимум нужно изучить за неделю

В данной настройке необходимо указать, какое минимальное количество регламентов сотрудник должен изучить за неделю.

## Коммуникатор

Настройки для отслеживания и обработки заторов в коммуникациях.

#### Укажите функцию для получения ежедневного отчета по заторам в коммуникациях

Позволяет задать, кто будет получать задачи с ежедневным отчётом о нарушениях по коммуникациям.

#### Укажите функцию хранителя системы

Указывает, кто в организации принимает информацию от коммуникаторов по нарушениям коммуникаций.

## Оплата

Автоматизация задач по выставлению и контролю оплаты.

#### Отправлять счёт на оплату в коммуникации до окончания оплаченного периода

Позволяет включить напоминание об оплате тарифа за 14 дней до окончания текущего периода.

#### Укажите функцию, которая отвечает за оплату системы

Устанавливает функцию, сотрудники которой получат задачу об оплате тарифа.

## Сотрудники

Блок управления напоминаниями о днях рождения сотрудников.

#### Укажите, кому будет назначена задача с напоминанием о дне рождении сотрудника

Настройка для назначения ответственного за создание задачи с поздравлением.

#### Напомнить о дне рождении сотрудника

Указывает, за сколько дней до события система отправит напоминание.
