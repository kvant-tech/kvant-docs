# Настройки

В этом разделе описаны все настройки Приложения Квант, которые позволяют адаптировать систему под нужды вашей организации.

## График работы организации

**1. По умолчанию, раздел "Настройки" доступен только руководителю компании.**

Чтобы получить доступ к разделу "настройки", необходимо запросить на функцию разрешение "Показывать меню настроек" у руководителя компании.

**2. Чтобы изменить график работы организации, необходимо перейти в раздел "настройки".**

**3. Во вкладке "Основные", включите опцию "Указать график работы организации".**

В поле ниже, "График работы организации" - укажите рабочие дни и время работы.

Кнопка "Все будни по одному графику" объединяет все будни дни, если галочка включена.

Если Вам необходимо указать время работы отдельно для будних дней, отключите галочку в поле "Все будни по одному графику" и оформите персонально время работы для каждого дня.

Если "график работы организации" сформирован, обязательно нажмите кнопку "сохранить" внизу страницы, чтобы сохранить внесённые изменения.

## График работы отдельных функций

1. По умолчанию раздел "настройки" доступен только руководителю компании.

Чтобы получить доступ к разделу "настройки", необходимо запросить на свою функцию разрешение "Показывать меню настроек" у руководителя компании.

2. Чтобы сформировать отдельный график для функций, необходимо перейти в раздел "настройки".

![](/from-carrot-images/34467-1743273942801-m83htyw0.png)

3. Во вкладке "Основное", нажмите на кнопку "Добавить отдельный график для функций".

![](/from-carrot-images/34467-1743274062307-d43ss5vp.png)

4. Заполните наименование графика работы

![](/from-carrot-images/34467-1743274182376-za1ej65j.png)

5. Нажмите кнопку "добавить функцию" и выберите из списка одну или несколько функций, которым необходимо указать график работы.

![](/from-carrot-images/34467-1743274289078-12eakmb9.png)

6. Укажите дни и время работы функции.

Опция "Все будни по одному графику" объединяет все дни пн-пт.

![](/from-carrot-images/34467-1743274369555-e3ilgxa9.png)

Если Вам необходимо указать время работы отдельно для будних дней, отключите опцию "Все будни по одному графику" и укажите время работы на каждый день отдельно.

![](/from-carrot-images/34467-1743274463441-77uy9iir.png)

Если "отдельный график для функций" сформирован, обязательно не забудьте нажать кнопку "сохранить" внизу страницы.
