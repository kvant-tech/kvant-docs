# Настройки

Раздел "Настройки" предназначен для конфигурации ключевых параметров вашей организации в системе. Он включает в себя четыре подраздела — Основное, Дополнительные, Расширенные и Справочники, каждый из которых содержит важные параметры, влияющие на логику работы, доступы, коммуникации и автоматизацию процессов. Правильная настройка этих параметров позволяет обеспечить гибкость, прозрачность и эффективность функционирования организации.

## Содержание

- [Основное](/reference/settings/basic) - Ключевые настройки организации и базовые параметры работы системы
- [Дополнительные](/reference/settings/additional) - Расширенные настройки для персонализации работы
- [Расширенные](/reference/settings/advanced) - Профессиональные настройки для опытных пользователей
- [Справочники](/reference/settings/reference) - Уровни администратора и типы исполнителей
- [График работы](/reference/settings/work-schedule) - Настройка графиков работы организации и отдельных функций
