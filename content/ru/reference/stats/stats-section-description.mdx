# Внесение и просмотр показателей

Раздел «Статистика» предоставляет инструменты для работы с показателями и статистиками. В разделе представлены следующие вкладки:

## Вкладки раздела

![Вкладки раздела](/from-carrot-images/stats1.png)

### Мои

В этой вкладке отображаются показатели статистики по функции, к которой вы назначены. При добавлении статистики через функцию, можно отметить её как главную — такая статистика выделяется специальным значком.

При нажатии на значок вопроса открывается описание выбранной статистики.

Показатели заполняются вручную — для этого нужно выбрать временной интервал, за который ведётся статистика.

Также при нажатии на иконку «шестерёнка» можно включить отображение недельного тренда. Количество недель выбирается из выпадающего списка в зависимости от желаемого периода отображения.

### Сотрудники

Здесь отображаются статистики исполнителей, а также сотрудников, находящихся под определённой функцией в организационной структуре (ОФС). Статистики администраторов отображены не будут.

Данный раздел позволяет отслеживать как общую статистику по функции, так и показатели каждого специалиста по отдельности.

### Все

В этой вкладке собраны общие статистики из разделов «Мои» и «Сотрудники», предоставляя полный обзор по всем функциям.

### Добавить дашборд

Эта функция позволяет создавать доски (дашборды), на которые можно группировать статистики разных функций. Дашборды служат удобным инструментом для отображения всех важных показателей компании в одном месте.

Чтобы добавить статистику на дашборд, выберите необходимые статистики из функций, которые должны быть представлены на доске.

Статистики подфункций можно сортировать и менять последовательность их отображения.

Также доступна возможность отображать тренд за выбранное количество недель.

Визуализацию данных можно переключать между списком и сеткой.

При необходимости дашборд можно отредактировать — изменить название или удалить.
