# Сотрудники

*Управление персоналом и организационная структура*

Раздел "Сотрудники" предоставляет инструменты для управления персоналом в Приложении Квант. Здесь вы можете регистрировать новых сотрудников, настраивать графики работы, управлять персональными данными.

## Содержание раздела

- **[Управление сотрудниками](/reference/employees/employee-management)** - регистрация новых сотрудников и удаление сотрудников из системы
- **[Email как идентификатор](/reference/employees/email-identifier)** - правила работы с электронной почтой как уникальным идентификатором пользователя
- **[День рождения сотрудника](/reference/employees/birthday-notifications)** - настройка автоматических уведомлений о днях рождения
- **[Посменный график работы](/reference/employees/shift-work-setup)** - подробная инструкция по настройке посменных графиков для функций
