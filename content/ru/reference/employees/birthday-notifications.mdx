# Напоминание о дне рождения сотрудника

*Автоматическая отправка поздравлений с днём рождения*

Система поддерживает автоматическую отправку поздравлений с днём рождения. Для работы функции необходимо:

#### 1. Указать дату рождения сотрудника

В разделе Сотрудники → Список, при редактировании карточки сотрудника доступно поле Дата рождения. При отсутствии значения отображается статус Не указано. Поддерживается ввод дня, месяца и года.

#### 2. Настроить функцию для получения уведомлений

В разделе Настройки → Дополнительные → Сотрудники доступна настройка:
«Укажите, кому будет назначена задача с напоминанием о дне рождении сотрудника».
Выберите функцию (например, руководитель отдела), сотрудники которой будут получать напоминания.

#### 3. Логика назначения коммуникации

**Отправитель:** администратор указанной функции.
**Получатель:** исполнитель функции.

При отсутствии администратора отправитель — исполнитель.
При отсутствии исполнителя отправитель — администратор.
Если функция не указана или в ней нет сотрудников — коммуникация не создаётся.

#### 4. Настроить время напоминания

Ниже находится настройка «Напомнить за _ день».
По умолчанию установлено значение 1. Поддерживается значение 0 — отправка в день события. Слово день склоняется автоматически.

#### 5. Автоматическое создание коммуникации

Коммуникация создаётся в 10:00 по часовому поясу сотрудника.

**Формат текста:**
«[дата] празднует свой день рождения [ФИО сотрудника]! Не забудьте поздравить!»

При наличии нескольких дней рождения в один день формируются отдельные обращения для каждого сотрудника.
