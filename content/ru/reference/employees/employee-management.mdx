# Управление сотрудниками

*Регистрация новых сотрудников и удаление сотрудников из системы*

Для того, чтобы иметь возможность добавлять и удалять сотрудников в разделе "Сотрудники", у вас на функции должно быть включено разрешение "Управление сотрудниками". Разрешение на функцию необходимо запросить у владельца системы.

## Регистрация нового сотрудника

### Процесс регистрации

#### 1. Добавление сотрудника

Перейдите в раздел "Сотрудники", вкладка "список" и нажмите кнопку "Добавить сотрудника"

#### 2. Заполнение данных

Заполните актуальные данные сотрудника и нажмите кнопку "Добавить".

**Обратите внимание:** необходимо подтвердить, что у вас есть разрешение на обработку персональных данных сотрудника.

#### 3. Приглашение по email

После сохранения, сотруднику на указанную почту поступит письмо с приглашением. Адрес электронной почты, с которого поступит приглашение: notify@rsuquant.ru

Если письмо не отображается на электронной почте в папке "Входящие", проверьте папку "Спам"

#### 4. Автоматические задачи

При входе в систему, новому сотруднику будет назначено прохождение вводного обучения автоматически. После прохождения вводного обучения будут созданы задачи на заполнение личного профиля и подключение к телеграм боту для оповещений.

### Требования к регистрации

**Разрешения:** Для регистрации сотрудников необходимо разрешение "Управление сотрудниками" на функции

**Персональные данные:** Обязательно подтверждение разрешения на обработку персональных данных

**Email:** Уникальный email адрес для каждого сотрудника

## Удаление сотрудника

### Процесс удаления

#### 1. Переход в раздел "Сотрудники"

Перейдите в раздел "Сотрудники"

#### 2. Поиск сотрудника

Введите данные для поиска сотрудника в поисковой строке.

В поисковой строке поиск осуществляется по следующим данным: email, имя, фамилия или отчество, а также по номеру телефона

#### 3. Удаление сотрудника

Чтобы удалить сотрудника из системы необходимо нажать на три точки в конце строки (1). В выпадающем списке выберите **Удалить** (2).

**Примечание:** Также можно удалить несколько сотрудников сразу. Для этого выберите сотрудников которых хотите удалить с левой стороны и также нажмите удалить как при удалении одного сотрудника

#### 4. Настройка обработки коммуникаций

Система предлагает выбрать, что делать с коммуникациями сотрудника.

По умолчанию, выбрано значение "распределить коммуникации между сотрудниками организации".

При назначении нового сотрудника все нарушения по коммуникациям у уволенного сотрудника должны сброситься и создаваться по новой (датой создания коммуникации считать дату назначения этого сотрудника исполнителем/отправителем. При выборе данного варианта во всех коммуникациях уволенного сотрудника меняем отправителя/исполнителя на нового сотрудника, кроме коммуникаций на этапе "Выполнено".

В опции "Переносить крайний срок в коммуникациях с просроченным крайним сроком на" - укажите количество и значение(год, месяц, день, час, минута)

#### 5. Подтверждение удаления

Нажмите кнопку "Уволить".

**Примечание:** Если выбрано несколько сотрудников для удаления такое окно будет показываться на каждого сотрудника

### Важные требования

**ВАЖНО!** Для данных действий у вас на функции должно быть включено разрешение "Управление сотрудниками". Разрешения на функции выдает руководитель компании.
