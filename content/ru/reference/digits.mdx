# Сводка по сотрудникам

Раздел содержит краткую сводку по текущему составу команды. Здесь отображаются:

- общее количество сотрудников в компании;
- количество руководителей (администраторов функций);
- число сотрудников, которые заходили в систему сегодня;
- количество сотрудников, принятых на работу сегодня.

Руководители — это сотрудники, назначенные на позицию администратора на одной или нескольких функциях на ОФС.

Сводка позволяет оценить состав и активность сотрудников на текущий момент.
