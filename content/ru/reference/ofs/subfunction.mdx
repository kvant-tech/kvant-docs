# Как создать подфункцию

В этой инструкции вы узнаете, как создать подфункцию в разделе "Функции".

## Требования

Данный функционал доступен по разрешению **"Редактирование ОФС"** на одной из
своих функции. Разрешение можно запросить у "руководителя" организации.

**Важно:** На функции у которой имеются подфункции не может быть исполнителей -
эта функция является административной. При попытке добавить подфункцию на
функцию где есть исполнители появится предупреждение.

## Создание подфункции

Для добавления подфункции на ОФС (Организационно-функциональной схему)
необходимо:

1. Нажать на три точки на той функции, к которой хотите добавить подфункцию
2. Выбрать "Добавить подфункцию"

После этого откроется всплывающее окно с настройками новой функции.

## Настройки подфункции

### Основные настройки

1. **Название функции** (обязательное поле) - название подфункции
2. **Ожидаемый конечный результат** - результат, который ожидается от всех
   сотрудников на функции
3. **Выделять цветом** - возможность выделить функцию цветом для визуального
   отображения на схеме
   - После активации появится белый квадратик
   - При нажатии откроется палитра для выбора цвета

### Настройки отображения

4. **Отрисовка** - управление отрисовкой функций на ОФС:
   - Автоматическая - функция выделяется в отдельный столбец справа при наличии
     подфункции
   - Вниз - функция не выделяется в отдельный столбец, подфункции отображаются
     ниже
   - В столбец - функция выделяется в отдельный столбец даже без подфункций

### Управление персоналом

5. **Уровень Администратора** - название уровня руководителя функции:

   - директор
   - заместитель директора
   - руководитель департамента/отделения
   - начальник отдела

6. **Добавить тип исполнителей** - возможность объединения функций в группы:
   - На одну функцию можно добавить несколько типов исполнителей
   - Позволяет настраивать повторяющиеся задачи и распространять регламенты на
     группы функций

### Дополнительные настройки

7. **Посменная работа** - отметка о посменном графике работы сотрудников функции

8. **Скрыть ОФС выше этой функции** - ограничение видимости ОФС:

   - При включении сотрудники видят только эту функцию и подфункции
   - Остальные функции отображаются серым цветом и недоступны для редактирования
   - Не применяется, если сотрудник имеет доступ к другим функциям без этого
     ограничения

9. **Наследовать регламенты** - управление получением регламентов
   администратором:
   - Включено - администратор получает регламенты с подфункций
   - Выключено - администратор не получает регламенты с подфункций

## Ограничения

При попытке добавить подфункцию на функцию с исполнителями система выдаст
ошибку:

> На функции у которой имеются подфункции не может быть исполнителей. Только
> администраторы! Исполнителями административной функции являются все
> исполнители с подфункций.
