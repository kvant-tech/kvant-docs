# Разрешения на функцию

## Как выдать разрешения на функцию

В этой статье вы научитесь выдавать разрешения на функцию сотрудников.

Разрешения нужны для расширения функционала сотрудника, который стоит на определенной функции. Разрешения выдаются на функцию.

Разрешения бывают 2-х типов: для администраторов и исполнителей функции.

## Процесс выдачи разрешений

### 1. Переход к управлению функцией

Переходим в раздел "Функциональная схема" и выбираем функцию, на которую хотим выдать разрешения. Нажимаем правую кнопку мыши, чтобы открыть меню функции и пункт "Открыть управление функцией".
{/* ![Переход к управлению функцией](/images/34467-1743502219929-6p9e3w68.png) */}

### 2. Настройка разрешений

В управлении функцией переходим на вкладку "Разрешения".

{/* ![Настройка разрешений](/images/34467-1743502278051-0buy1adr.png) */}

### 3. Сохранение изменений

Во вкладке "Разрешения" указаны разрешения для администраторов и для исполнителей данной функции, после внесения нужных разрешений необходимо нажать кнопку "сохранить" внизу страницы.

### 4. Справочная информация

Для получения справочной информации о каждой из настроек, необходимо напротив каждого разрешения нажать на "Вопросительный знак".

## Категории разрешений

Все разрешения сгруппированы в списке по разделам для удобства.

### ОФС

- **Редактирование ОФС** - предоставляет возможность редактировать ОФС (создавать новые функции, удалять существующие, перемещать функции)
- **Управление сотрудниками на ОФС** - позволяет ставить и снимать сотрудников на функции на ОФС

### Повторяющиеся задачи

- **Управление повторяющимися задачами на всех функциях** - дает возможность управлять повторяющимися задачами (создавать и редактировать) на всех функциях
- **Разрешить удаление повторяющихся задач на всех функциях** - дает возможность удалять и ставить на паузу повторяющиеся задачи на всех функциях, созданные от любых сотрудников

### Коммуникации

- **Разрешить изменение отправителя в коммуникации** - позволяет изменять отправителя в коммуникациях
- **Разрешить изменение получателя в коммуникации** - позволяет изменять получателя в коммуникациях
- **Показывать массовые задачи** - в коммуникации появляется вкладка Массовые задачи и возможность создавать такие задачи

### Отчеты

- **Показывать отчет по регламентам**
- **Показывать отчет по статистикам**
- **Показывать отчет по коммуникациям**
- **Показывать отчет по коммуникатору**
- **Показывать отчет по заторам**
- **Показывать отчет список коммуникаций**
- **Показывать отчет список ПЗ**

### Разделы

- **Показывать меню настроек**
- **Показывать раздел Тарифы**
- **Показывать раздел Табель рабочего времени**
- **Показывать Бизнес-процессы**

### Управление

- **Управление разрешениями** - позволяет редактировать разрешения на функциях
- **Управление сотрудниками** - позволяет добавлять и удалять сотрудников в разделе "Сотрудники"
- **Разрешить видеть документы сотрудников** - позволяет видеть прикрепленные документы сотрудников в разделе Сотрудники
- **Управление описанием функций** - дает возможность редактирования описания функции
- **Разрешить изменение графика работы** - позволяет редактировать посменный график работы для сотрудников в разделе "Сотрудники"
- **Разрешить изменение бизнес-процессов** - Открывает доступ к просмотру и редактированию бизнес-процессов
- **Разрешить удаление шаблонов графиков работы** - Позволяет удалять созданные шаблоны графиков работы в разделе "Сотрудники"
- **Разрешить просматривать календарь других сотрудников** - Позволяет просматривать календари других сотрудников. У пользователей с этим разрешением появляется фильтр в календаре, в котором можно выбрать календарь какого сотрудника необходимо отобразить
- **Разрешить массовое закрытие коммуникаций** - Позволяет массово закрывать коммуникации в отчете Список коммуникаций

### Регламенты

- **Проверка регламента** - предоставляет возможность проверять регламенты, при наличии регламентов на одобрение, будет отображаться вкладка "Проверить", если регламентов на проверку нет, вкладка не отображается
- **Одобрение регламента** - предоставляет возможность одобрять регламенты, при наличии регламентов, которые требуют одобрения, будет отображаться вкладка "Одобрить", если регламентов на одобрение нет, то вкладка не отображается
- **Включать публичный доступ к регламентам** - дает разрешение включить публичный доступ к регламентам
- **Показывать сотрудников, изучивших регламент** - дает возможность просматривать список сотрудников, которые изучили регламент
- **Просмотр всех регламентов по функциям** - открывает доступ к просмотру всех регламентов по всем функциям компании
- **Разрешить изменения в метках** - позволяет создавать, редактировать и удалять метки в регламентах
- **Разрешить редактировать регламенты на функциях** - данное разрешение позволяет через "ОФС" добавлять или удалять регламенты на функциях

### Статистики

- **Просматривать все статистики** - дает разрешение на просмотр всех статистик
- **Управление статистиками** - позволяет управлять статистиками

## Важно

После внесения любых изменений нажмите кнопку "Сохранить".
